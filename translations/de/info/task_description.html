Nicola macht einen dringend benötigten Urlaub. Er wird als Rucksacktourist einige weniger bekannte Länder bereisen. Jedes hat seine eigene, einzigartige Währung.

Beim Einkaufen möchte Nicola so wenig Münzen wie möglich verwenden. In Outer Leftopia gibt es zum Beispiel Münzen mit einem Nennwert von 1, 3 und 5 Schilling. Er möchte ein Souvenir kaufen, das 13 Shillings kostet. Dafür kann er zwei 5-Schilling-Münzen und eine 3-Schilling-Münze verwenden.

Sie können davon ausgehen, dass Nicola Zugang zu einer unendlichen Anzahl von Münzen jedes Nennwerts hat. (Er hat ein großes Bankkonto und Zugang zu einem Geldautomaten).

Eingabe: Zwei Argumente. Das erste Argument ist ein int: der Preis des Kaufs. Das zweite Argument ist eine Liste von Ints: die Stückelungen der verfügbaren Münzen.
